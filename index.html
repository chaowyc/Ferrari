Welcome to the project page of the FERRARI reachability index.

FERRARI (Flexible and Efficient Reachability Range Assignment for Graph Indexing) is a fast and compact, memory-resident index structure for answering reachability queries over large graphs.

Introduction

A reachability query for a pair of vertices (u,v) returns a positive answer if the graph under consideration contains a directed path starting at vertex u and ending at vertex v, and a negative answer otherwise.
